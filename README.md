# Fundación Andrés Grajales - Web Platform

## Setup Instructions

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Asset Configuration:**
    Ensure your `images/` folder is moved inside a new `public/` folder for Vite to serve them correctly.
    
    *Structure should look like:*
    ```
    /public
      /images
        logo_fag.png
        fag1.png
        fag2.png
        ...
    ```

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

## Design Philosophy: Humanistic Engineering

- **Typography:** Playfair Display (Serif/Human) + JetBrains Mono (Engineer).
- **Colors:** Slate 900/50 (Binary contrast) + Accents (Growth).
- **Layout:** Mathematical grids and Golden Ratio spacing.
